# Launch Checklist

## Before Posting

- Confirm GitHub Release exists for the intended version.
- Confirm README install instructions match current publishing status.
- Confirm CI is green.
- Confirm screenshots or console examples contain no secrets, hostnames, private paths, or tokens.
- Confirm PyPI is not mentioned as available unless it is actually published.

## Content Points

- Local AI debugging is painful.
- InferDoctor gives one-command diagnosis and Top Fixes.
- v0.3 adds setup-assistant workflow: recommend, stack plan, templates, validation, and model fit.
- v0.4 preview focuses on dry-run bootstrap, smoke tests, workflow guides, and better beginner UX.
