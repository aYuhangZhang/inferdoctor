# GitHub Release Copy: v0.3.0

InferDoctor v0.3.0 turns the project into a local AI stack doctor and setup assistant.

It helps developers diagnose local AI setup problems and move toward a working starter app:

- `inferdoctor` health dashboard
- Top Fixes for common local AI failures
- `inferdoctor recommend`
- `inferdoctor stack plan`
- `inferdoctor template create`
- `inferdoctor template validate`
- `inferdoctor model fit`

v0.3 is still lightweight and safe by default. It does not install runtimes, download models, run inference, or modify system settings.

Install from GitHub until PyPI publishing is completed:

```bash
python -m pip install "git+https://github.com/anguoyang/inferdoctor.git@v0.3.0"
```
