# InferDoctor Promotion Assets

Public-facing launch materials for InferDoctor.

Core message:

InferDoctor diagnoses your local AI stack and helps you start building local AI apps.

Pain point:

Local AI is powerful, but setup is confusing. Ports, CUDA, endpoints, runtimes, model size, templates, and app scaffolding all get in the way.

InferDoctor helps with:

- health check
- Top Fixes
- capacity preview
- recommendations
- starter templates
- template validation
- dry-run setup plans

Keep claims practical. Do not claim model downloads, runtime installation, production deployment automation, or benchmark accuracy.

Default install command for public launch materials:

```bash
pip install inferdoctor
```
