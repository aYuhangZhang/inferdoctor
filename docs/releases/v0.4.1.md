# InferDoctor v0.4.1 - First PyPI Release

InferDoctor v0.4.1 is the first PyPI release of InferDoctor. The package name is `inferdoctor`, and the default install path is now:

```bash
pip install inferdoctor
```

InferDoctor is a local AI stack doctor and setup assistant. It helps developers diagnose local AI runtime issues, understand what their machine can realistically run, and start from practical local AI app templates.

## What Is Included

- One-command local AI health dashboard with `inferdoctor`.
- Setup assistant workflow for choosing a practical next step.
- Starter template catalog and project generation.
- `inferdoctor stack plan` for beginner-friendly setup plans.
- `inferdoctor template validate` for generated project checks.
- `inferdoctor template smoke-test` for safe dry-run template startup checks.
- `inferdoctor model fit` for heuristic model-size and VRAM guidance.

## Quick Start

```bash
pip install inferdoctor
inferdoctor
inferdoctor template list
inferdoctor stack plan --goal customer-service --vram 24
inferdoctor template create customer-service --output ./customer-service-demo
inferdoctor template validate ./customer-service-demo
inferdoctor template smoke-test ./customer-service-demo
```

## Safety Defaults

InferDoctor remains lightweight and safe by default:

- It does not download models.
- It does not install Ollama, vLLM, SGLang, Xinference, CUDA, Docker, or other runtimes.
- It does not run model inference during diagnosis, validation, smoke testing, or stack planning.
- It does not modify system settings.
- Template generation writes only to the output directory chosen by the user.

## Known Limitations

- Recommendations and model-fit estimates are heuristics, not benchmarks.
- Template smoke tests check safe startup paths only; they do not prove a live model endpoint is healthy.
- Generated templates are starter examples, not production-ready deployments.

## Development Install

For development from the repository:

```bash
git clone https://github.com/anguoyang/inferdoctor.git
cd inferdoctor
python -m pip install -e ".[dev]"
```
