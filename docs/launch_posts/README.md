Positioning to keep consistent:

- InferDoctor diagnoses local AI inference stacks.
- It is not a model recommender.
- It does not download models, run inference, benchmark hardware, or modify system settings.
- It should remain lightweight, read-only by default, and useful on CPU-only machines.
